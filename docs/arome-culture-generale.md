## Techniques de Formulation

### Équilibre Top / Middle / Base
| Position | Caractéristiques | Exemples de molécules |
|----------|------------------|----------------------|
| **Top (tête)** | Volatile, impact immédiat | Aldéhydes légers, esters courts |
| **Middle (cœur)** | Corps de l'arôme | Lactones, alcools terpéniques |
| **Base (fond)** | Rémanence, fixation | Vanilline, muscs, lactones lourdes |

### Principes de Base
1. **Commencer simple** - 3-5 ingrédients pour le squelette
2. **Construire par couches** - Top → Middle → Base
3. **Utiliser des accords** - Combinaisons éprouvées
4. **Équilibrer les seuils** - Ajuster selon les thresholds
5. **Tester en application** - Un arôme change selon le support

### Accords Classiques
| Accord | Molécules | Résultat |
|--------|-----------|----------|
| Beurré | Diacétyl + Acétoïne + δ-Décalactone | Beurre crémeux |
| Fraise | Furaneol + γ-Décalactone + cis-3-hexénol | Fraise mûre |
| Vanille riche | Vanilline + Éthylvanilline + Coumarine | Vanille complexe |
| Agrume frais | Limonène + Linalol + Citral | Citrus frais |
| Torréfié | Pyrazines + Furfurylthiol + Guaïacol | Café/cacao |

## Chimie des Arômes

### Terpènes
Les terpènes sont des composés organiques produits par de nombreuses plantes.

**Exemples de terpènes :**
- Linalol
- Limonène
- Myrcène

**Le maltol n'est PAS un terpène** - c'est une lactone utilisée comme exhausteur de goût sucré.

### Formulation d'une Note Beurrée
Pour obtenir un bouquet "beurré", utiliser :
- **Diacétyl** (2,3-butanedione) - composé principal du goût beurre
- **Acétoïne** (3-hydroxy-2-butanone) - renforce la note beurrée

### 2,4-Décadiénal
Le 2,4-décadiénal apporte une **note grasse** (gras/friture) dans un arôme. On le retrouve notamment dans les huiles chauffées.

### Estragol
**Synonyme : Méthyl Chavicol**

Composé aromatique présent naturellement dans l'estragon, le basilic et l'anis.

## Stabilité des Arômes

### Facteurs de Dégradation
| Facteur | Molécules sensibles | Protection |
|---------|---------------------|------------|
| Oxydation | Aldéhydes, terpènes | Antioxydants (BHT, tocophérols) |
| Lumière | Aldéhydes, citrus | Emballage opaque |
| pH acide | Acétals, certains esters | Tampons |
| pH basique | Esters, lactones | Tampons |
| Chaleur | Thiols, esters volatils | Stockage froid |

### Molécules Particulièrement Instables
- **Citral** : Cyclise en pH acide
- **Aldéhydes** : Oxydation, formation d'acétals
- **Thiols** : Oxydation en disulfures
- **Terpènes** : Isomérisation, oxydation

## Réglementation des Arômes

### Définition d'un Arôme Naturel de X
Un arôme naturel "de X" doit contenir **au moins 95%** de sa partie aromatisante issue de la source X nommée.

### Catégories Réglementaires (EU)
| Catégorie | Définition |
|-----------|------------|
| **Arôme naturel** | 100% substances aromatisantes naturelles |
| **Arôme naturel de X** | ≥95% de la source X |
| **Arôme naturel de X avec autres arômes naturels** | Part significative de X |
| **Arôme** | Peut contenir substances identiques au naturel ou artificielles |

### FEMA GRAS
Le système FEMA (Flavor and Extract Manufacturers Association) classe les substances comme **GRAS** (Generally Recognized As Safe) pour l'usage aromatique aux États-Unis.

### Les 5 Saveurs de Base
| Saveur | Récepteurs | Exemples de stimuli |

### Sensation Acide
Parmi les composés courants, l'**acide citrique** (Citric Acid) donne une sensation acide.

- Saccharose → sucré
- Chlorure de sodium → salé
- Acide glutamique → umami

## Aromates vs Épices

### Classification
**Aromates** (herbes aromatiques) :
- Basilic
- Laurier
- Romarin

**La cardamome est une épice**, pas un aromate. Les aromates sont généralement des parties vertes de plantes (feuilles), tandis que les épices proviennent d'autres parties (graines, écorces, racines).
